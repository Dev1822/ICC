# ICC Website Clone

A simple **ICC (International Cricket Council) website clone** built using **HTML and CSS**.  
This project recreates the look and feel of the ICC website with multiple pages for matches, rankings, and news.

🌐 **Live Demo:**  
https://devpatel-iccclone.netlify.app/

---

## Features

- ICC themed UI
- Multiple pages (Matches, Rankings, News)
- Custom CSS styling
- Responsive layout
- Header and Footer navigation

---

## Tech Stack

**Frontend**
- HTML5
- CSS3

**Tools**
- Git
- GitHub
- Netlify (Deployment)
